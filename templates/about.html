About MyTodo

FlowTodo is a modern task management web application designed to help users organize daily activities efficiently. The system allows users to create, update, and manage tasks in a simple and productive way.

Project Mission

In today's fast-paced digital world, productivity tools are essential. FlowTodo was developed to provide a clean and simple platform for managing personal tasks without unnecessary complexity. The goal of this project is to combine simplicity with functionality, allowing users to focus on their work instead of managing complicated software.

⚡ Fast

Lightweight backend powered by Flask for fast performance.

🔒 Secure

Data stored safely using SQLite database.

Technologies Used

  • Python – Core backend programming language
  • Flask – Lightweight web framework used for routing and application logic
  • Flask-SQLAlchemy – ORM used to interact with the database
  • SQLite – Database used for storing todo tasks
  • HTML5 – Used to structure the web pages
  • CSS3 – Used to design and style the user interface
  • JavaScript – Adds dynamic behaviour to the application
  • Bootstrap – Responsive front-end framework for layout and design

Core Features

  • Add new tasks easily
  • Update existing tasks
  • Delete completed or unnecessary tasks
  • Store data persistently using SQLite
  • Responsive interface compatible with desktop and mobile devices
  • Simple and user-friendly task management system

Developer

This project was developed as a learning project to understand modern web development concepts using Flask. It demonstrates important features such as routing, template rendering, database integration, and CRUD (Create, Read, Update, Delete) operations.

The main objective of developing FlowTodo is to practice full-stack development using Python and Flask while building a practical productivity tool.
